# DynaCASE

See https://dynacase.github.io/

## Installation

Download Pharo 5 (4 maybe also works) image and run the following

```
Metacello new
    baseline: 'DynaCASE';
    repository: 'github://dynacase/dynacase/repository';
    load.
```
